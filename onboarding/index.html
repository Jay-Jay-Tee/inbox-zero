Set up InboxZero AI

Two quick steps to connect your own Gemini API key and unlock AI summaries, smart spam detection, and your inbox dashboard.

Step 1 – Create a Gemini API key

  1. Go to Google AI Studio – API Keys .
  2. Sign in with the same Google account you use for Gmail.
  3. Click “Create API key” and copy the key that is shown.

Keep this key secret. It lives only in your browser via Chrome’s extension storage.

Step 2 – Paste the key into the extension

  1. In Chrome, click the Extensions icon in the toolbar and pin InboxZero AI if it isn’t already.
  2. Click the InboxZero AI icon to open the popup.
  3. In the “Enter API Key” area at the bottom:
    • Paste your Gemini key into the input box.
    • Click the ✓ button to save it.
    • The key display will turn green once saved successfully.

You can change or remove the key any time from the popup.

What you can do now

  • Summarize today’s emails from the popup dashboard.
  • Archive newsletters and clean noise in a click.
  • See real metrics on spam caught, time saved, and weekly volume.
  • Get per‑email AI summaries, categories, and spam scores inside Gmail.

You’re ready to go

Once your Gemini key is saved, open Gmail and look for the InboxZero AI buttons above any email, plus the new dashboard in the popup.
